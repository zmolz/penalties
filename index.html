date_pat = re.compile(r'(\d\d?)(\d\d)')

def parse_date(d):
    match = re.match(date_pat, str(d))
    month = '0'+match.group(1) if len(match.group(1)) == 1 else match.group(1)
    day = match.group(2)
    return f'{month}-{day}'

# create a closure with the regex and the dictionary and this function
def matchup_parser(matchup):
    match = re.match(matchup_pat, matchup)
    away = to_abv[match.group(1)]
    home = to_abv[match.group(2)]
    return home, away

# closure
def penalty_parser(penalty):
    pen_on = re.search(penalty_on_pat, penalty).group(1)
    pen = re.search(penalty_pat, penalty).group(1)
    pen = (pen.split(','))[-1]
    return pen_on, pen

# closure
def time_parser(time):
    time_match = re.match(time_pat, time)
    quarter = time_match.group(1)
    time = time_match.group(2)
    return quarter, time


for ref, games in ref_to_games.items():

As we look at the graph, we see something somewhat unexpected - as your team gets more penalties called on them, your team is expected to win more. The gamedata table includes information on games not just in the regular season, but in the playoffs as well, so those teams in the playoffs are getting more penalties called against them than other teams, and getting more wins as well. This could be a source of error. To adjust for this, we have to use a more reliable variable - win percentage would be a better gauge.

Analysis and Hypothesis Testing¶

To answer a question, we must first phrase it in a way such that we can test it in a way such that we will be able to definetively take something away once we conduct our experiment. i.e, no open ended questions that could result in 'Maybe'. We do this by proposing a null hypothesis - a baseline decision problem, the case in which there is no significant difference in the data for the question proposed.

To go back to our earlier question of Do the refs in the NFL favor the Vegas-favorited team?, the null hypothesis would be The refs show no favoritism for the Vegas-favorited team. So to start to answer our question, we must do a litle more data analysis.

We will calculate how many penalties were called on the favorited team vs underdog

In math, a statistical outlier is something that we define as outside 3 standard deviations of average. Looking at the graph, we can see that we have no statistical outliers in our dataset. Thus, we accept our null hypothesis of impartial referees.
